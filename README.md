# Majestic Pride Casino Newsletter Web App

This project is an online newsletter platform for Majestic Pride Casino, Goa's largest offshore casino. It features:

- Public newsletter page showing news from latest to oldest
- Each news article can include text, images, and an embedded live stream (e.g., YouTube Live)
- Admin portal for adding, editing, and deleting news articles (with authentication)
- Node.js + Express backend, React frontend, and SQLite database

## Folders
- `backend/` — Node.js + Express API and SQLite database
- `frontend/` — React app for public and admin interfaces
